¿Qué es una Apply?

Una "apply" (o solicitud) es tu manera formal de expresar interés en unirte al equipo del servidor Infinix. Implica responder reflexivamente a una serie de preguntas diseñadas para ayudarnos a entender tu personalidad, habilidades, dedicación e idoneidad para el rol deseado.

Requisitos Generales

  • Debes tener una cuenta de Discord activa y ser parte de nuestra comunidad.
  • Demostrar respeto, madurez y una actitud positiva.
  • Poseer un buen entendimiento de las mecánicas de Minecraft y la etiqueta del servidor.
  • Estar dispuesto/a a aprender, adaptarte y contribuir constructivamente al ambiente del servidor.
  • Puede aplicar una edad mínima para ciertos roles (detalles en las solicitudes específicas).

Herramientas Necesarias (General)

  • Una conexión a internet estable.
  • Una copia legítima de Minecraft Java Edition.
  • El cliente de escritorio o móvil de Discord.
  • Un micrófono funcional es altamente recomendado, especialmente para roles de Staff en adelante.
